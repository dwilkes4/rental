# Rental Application


## Database
Verify that:
    1. The following database properties are in your 'gradle.properties' file:
        i. `DB_ADMIN`
        ii. `DB_ADMIN_PWD`
        iii. `DB_NAME`
        iv. `DB_APP_USER`
        v. `DB_APP_USER_PWD`
    2. You are running on a Linux based system

Run:
    `./gradlew baseline`
