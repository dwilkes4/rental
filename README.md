# Rental Application


## Database
Verify that:

    1. The following database properties are in your 'gradle.properties' file:

        1. `DB_ADMIN`

        2. `DB_ADMIN_PWD`

        3. `DB_NAME`

        4. `DB_APP_USER`

        5. `DB_APP_USER_PWD`

    2. You are running on a Linux based system

Run:

    `./gradlew baseline`
