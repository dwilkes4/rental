# Rental Application


## Database

Verify that\:

    1. The following database properties are in your 'gradle.properties' file\:
        1. _DB_ADMIN_
        2. _DB_ADMIN_PWD_
        3. _DB_NAME_
        4. _DB_APP_USER_
        5. _DB_APP_USER_PWD_

    2. You are running on a Linux based system

Run:

    ./gradlew baseline
